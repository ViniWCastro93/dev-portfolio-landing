# Dev Portfolio Landing

Primeiro projeto de portfólio desenvolvido durante minha transição de carreira para Desenvolvimento Frontend.

## 📌 Sobre o projeto

Esta é uma landing page simples criada para apresentar meu perfil como desenvolvedor em formação e centralizar links profissionais.

O objetivo é demonstrar conhecimentos iniciais em estruturação de páginas web utilizando HTML e CSS.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## 🎯 Objetivo

Este projeto faz parte da construção do meu portfólio como desenvolvedor Frontend.

A ideia é evoluir progressivamente com novos projetos, aplicando conceitos de desenvolvimento web moderno.
